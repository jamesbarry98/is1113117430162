Infrastructure as a Service - Amazon

Infrastructure as a Service(IaaS) can be classified as Public, Private or Hybrid. Public Cloud is Infrastructure that consists of shared resources among a variety of users. Private Cloud exists and operates on a private network. Hybrid cloud is a combination of the two.


IaaS contains the building blocks for cloud IT and provides access to networking features, computers and data storage space.
